    체크리스트 및 완료 정책

    TL;DR

    • 포맷: PASS / FAIL / NA 3단계 선택
    • 강제성: 필수 항목 미입력 시 제출 불가, FAIL 선택 시 메모 입력 필수, 고객 서명 필수

    🛠️ 1. 설치(INSTALL) 체크리스트

    ✅ 필수 항목 (12개)

    ID 점검 항목 검증 기준 / 비고
    S01 협의된 설치 위치 확인 고객 요청 위치와 실제 설치 위치 일치 여부
    S02 전원 공급 상태 점검 콘센트, 차단기, 정격 전압 연결 확인
    S03 설치 공간 간섭 확인 통행 방해 여부, 고정 공간 및 안전 거리 확보
    S04 제품 모델 일치 확인 주문/지시 내용과 실물 모델 일치 여부
    S05 시리얼 번호 확인 실물 기기 시리얼 번호 수집 및 일치 여부
    S06 제품 외관 상태 확인 파손, 심각한 오염, 스크래치 유무
    S07 고정 상태 확인 브라켓, 거치대 등 고정 장치의 견고함
    S08 수평 및 정렬 상태 육안 기준 수평 및 각도 정렬 완료
    S09 전원 인가 및 동작 확인 부팅 정상 여부 및 기본 동작 테스트
    S10 에러 코드 및 알림 확인 이상 알림 표시 여부 및 조치 완료
    S11 사용자 교육 실시 기본 사용법 및 주의사항 안내 완료
    S12 고객 서명 완료 최종 승인 서명 (서명자 성명 포함)

    FAIL 시 메모 가이드: 원인 / 현장 조치 내용 / 잔여 이슈 / 고객 안내 사항

    🔧 2. AS(SERVICE) 체크리스트

    ✅ 필수 항목 (10개)

    ID 점검 항목 검증 기준 / 비고
    A01 접수 증상 재확인 고객 상담 내역과 실제 증상 대조 확인
    A02 대상 모델 확인 유지보수 대상 기기 모델 확인
    A03 시리얼 번호 확인 유지보수 대상 기기 시리얼 매핑
    A04 기본 연결 상태 점검 전원선, 통신선 등 기본 인프라 체크
    A05 원인 특정 및 재현 증상 재현 여부 확인 (미재현 시 사실 기록)
    A06 수리 및 조치 수행 부품 교체, 설정 변경, 클리닝 등 완료
    A07 조치 결과 검증 증상 해소 및 정상 동작 여부 최종 확인
    A08 조치 설명 및 안내 처리 내용 및 향후 주의사항 설명
    A09 후속 조치 안내 추가 부품 수급이나 방문 필요성 안내
    A10 고객 서명 완료 최종 승인 서명 (서명자 성명 포함)

    FAIL 시 메모 가이드: 증상 원인 추정 / 조치 내용 / 테스트 결과 / 추후 계획

    📸 3. 권장 항목 (공통)

    • [ ] 설치/조치 전 사진: 현장 상태 파악용 (1장 이상 권장)
    • [ ] 설치/조치 후 사진: 최종 결과 증빙용 (1장 이상 권장)
    • [ ] 특이사항: 현장 출입 제한, 환경적 특이점 등 메모

    ⚖️ 4. 완료 및 운영 정책

    🚩 FAIL 처리 정책

    • 필수 항목이 FAIL이어도 제출은 가능합니다. (현실적인 현장 상황 고려)
    • 단, FAIL 사유 메모가 반드시 입력되어야 하며, 필요 시 '후속 조치 필요' 플래그를 활성화합니다.

    🏁 완료 제출 최소 조건 (MVP 기준)

    1. 모든 필수 항목에 대한 응답 완료 (PASS / FAIL / NA)
    2. 고객 서명 데이터 저장 완료
    3. 사진(전/후) 업로드: 토글 OFF 기본 (Admin 설정으로 필수 변경 가능)
    4. 조직/팀 기본정보(OrgProfile/TeamProfile)는 완료 제출 요건 아님 (Phase 1) — 별도 화면에서 관리
