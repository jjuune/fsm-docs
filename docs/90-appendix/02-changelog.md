## v0.1 (2025-02-20)
### ✨ 주요 업데이트
- **문서 구조 체계화:** 숫자 프리셋 도입 및 폴더링 구조 확립 (00~90)
- **전체 문서 리뉴얼:** 총 20개 Markdown 파일의 가독성 개선 및 표 형식 도입
- **기술 스펙 확정:** API 명세, CRUD 매트릭스, 상세 UI 텍스트 목업 수립
- **데이터 모델 설계:** Mermaid 기반 ERD 및 필드 사전 v0.1 작성
- **GitHub 연동:** 자동 배포 워크플로우(`deploy.yml`) 및 MkDocs 환경 구축
