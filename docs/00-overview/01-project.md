# 프로젝트 정의

## TL;DR
- **Phase 1 목표:** 설치·AS 내부 운영을 디지털화하여 처리 속도, 추적성, 증빙 품질을 획기적으로 개선합니다.

---

## 🚀 프로젝트 개요 (Phase 1)
**프로젝트 명:** 공간살균기 설치·AS 내부 운영 디지털화 프로젝트

### 🎯 1. 프로젝트 목적
본 프로젝트는 기존의 수작업 및 분산 관리 체계를 디지털로 전환하여 다음을 달성하는 것을 목적으로 합니다.
- 현장 증빙(서명·사진·문서)의 **표준화**
- 본사–팀–기사 간 작업 흐름의 **통합 관리**
- 내부 운영 시스템의 **체계화**

### ⚠️ 2. 문제 정의 (Problem Statement)
현재 시스템의 주요 페인 포인트(Pain Points):
- 설치·AS 증빙 문서의 누락 및 분산 관리
- 현장 작업 이력 조회의 어려움
- 팀 단위 배정 체계의 비표준화
- 작업 완료 후 수동 방식의 문서 전달
- 관리자의 실시간 모니터링 한계

### ✅ 3. 핵심 목표 (Phase 1 Objectives)
| 구분 | 상세 목표 |
| :--- | :--- |
| **운영 효율** | 작업 생성 → 배정 → 완료 흐름 표준화, 팀 단위 관리 체계 정립 |
| **증빙 체계** | 현장 서명 필수화, 문서 자동 생성 및 고객 대상 자동 발송 |
| **통제/조회** | 통합 이력 조회, 관리자 대시보드로 상태 가시화 |

---

## 📌 프로젝트 범위 (Scope)

### ✅ 포함 범위 (In-Scope)
- **권한 체계:** Admin / Team Manager / Technician
- **작업 관리:** WorkOrder 기반 설치·AS 프로세스
- **현장 증빙:** 체크리스트 및 디지털 서명 수집
- **자동화:** PDF 문서 생성 및 이메일/문자 발송
- **데이터:** 작업 이력 조회 및 기초 통계

### ❌ 제외 범위 (Out-of-Scope)
- 영업 관리 모듈 및 가격 정책/프로모션
- 정산 및 회계 시스템 연동
- 고급 재고 관리 시스템
- 외부 고객 전용 SaaS 기능

---

## 📈 성공 기준 (KPI)
- **문서 누락률:** 0% (디지털 증빙 강제화)
- **조회 효율:** 이력 조회 시간 50% 이상 단축
- **가시성:** 작업 상태 100% 실시간 파악
- **자동화:** 고객 문서 발송 자동화 100%

---

## 🗺️ 향후 로드맵
- **Phase 2 (내부 고도화):** 재고/시리얼 추적 고도화, 영업 모듈 통합
- **Phase 3 (외부 확장):** 파트너사 통합 운영, SaaS 모델 전환 검토
