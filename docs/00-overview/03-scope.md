# 프로젝트 범위 (In/Out Scope)

## TL;DR
- **Phase 1:** 핵심 설치·AS 프로세스 및 증빙 체계 구축에 집중합니다.
- **Phase 2 이후:** 재고 관리, 정산, 고급 통계 등 부가 기능을 단계적으로 도입합니다.

---

## ✅ 1. In-Scope (Phase 1 필수 구현)

### 🛠️ 업무 프로세스
- **WorkOrder 관리:** 고객/현장 정보 매핑, 요약/특이사항, 방문 일정 관리
- **배정 체인:**
  - 본사(Admin) ➔ 팀 배정
  - 팀 관리자(TM) ➔ 기사 배정
- **현장 수행 (M-02):** 실시간 체크리스트(Autosave), FAIL 메모, 고객 서명(이름 포함), 사진 업로드(전/후)

### 💻 운영 시스템
- **팀 관리 화면 (T-01~03):** 부서별 운영지표, 미배정 작업함, 퀵 배정 기능
- **본사 관리 화면 (A-02, 04):** 전체 미배정 목록 관리, 팀 변경/취소, 문서/발송 상태 모니터링
- **문서 자동화:** 완료 후 PDF 자동 생성 및 스토리지 저장 (또는 다운로드 링크 제공)
- **보안/권한:** 역할 기반(Admin/TM/T) 및 스코프(ORG/TEAM/SELF) 차단

### 🆕 관리 기능 (Phase 1 추가)
- **로그인/홈 (L-01 / H-01):** 역할별 랜딩 라우팅 (Admin → A-00, TM → T-00, Tech → M-01)
- **Admin 팀 관리 (A-01):** 팀 생성·수정·비활성화 CRUD-lite (팀명 immutable, 삭제 금지)
- **TM 기사 관리 (T-04):** 기사 초대·수정·비활성화 CRUD-lite (role=TECH 고정)
- **운영 통계 MVP (A-00 / T-00):** 팀·기사별 상태 카운트 카드 수준 (성과평가 아님)
- **조직/팀 기본정보 (OrgProfile / TeamProfile):** 회사·센터 기본 데이터 + 간단 편집 화면

## ❌ 2. Out-of-Scope (Phase 2 이후 보류)

| 카테고리 | 제외/보류 항목 | 예정 단계 |
| :--- | :--- | :---: |
| **데이터 삭제** | **Team/User Hard delete** (Phase 1은 비활성화만) | Phase 2 |
| **재고/물류** | 시리얼 단위 정밀 추적, 입출고 관리 | Phase 2 |
| **영업/정산** | 가격 정책/프로모션, 세금계산서, 결제 연동, 매출 통계 | Phase 2+ |
| **시스템 확장** | 멀티 테넌트(SaaS), 세부 Permission 테이블 관리 | Phase 3 |
| **기능 고도화** | 발송 템플릿 관리, **고급 리포트/성과 지표**, **PDF 사업자정보 자동삽입(서식 자동화 고도화)**, 오프라인 모드 | Phase 2 |
| **감사/이력** | **AuditLog 전체 노출**, 감사 리포트 내보내기 | Phase 2 |
| **알림/관제** | SLA 에스컬레이션, 푸시 알림, 슬랙 연동 | Phase 2 |

## 🎯 3. Phase 1 성공 체크리스트 (MVP 기준)
- [ ] 기사가 모바일 웹에서 체크리스트와 서명 제출까지 문제없이 완료했는가?
- [ ] 팀 관리자가 목록에서 즉시 기사를 배정할 수 있는가?
- [ ] 본사 관리자가 전체 작업 흐름을 실시간으로 확인 가능한가?
- [ ] 작업 완료 후 조회가 가능한 PDF 문서 결과물이 생성되는가?
