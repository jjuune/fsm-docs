    🚀 FSM 디지털 전환 프로젝트 (Phase 1)

    "모든 현장의 기록을 디지털로 연결하고, 신뢰를 데이터로 증명합니다."

    Welcome! 본 사이트는 공간살균기 설치·AS 내부 운영 디지털화를 위한 통합 프로젝트 문서 저장소입니다.
    수작업 기반의 운영 체계를 디지털로 전환하여 본사, 팀, 기사가 하나의 흐름으로 소통하는 스마트한 워크플로우를 지향합니다.

    🗺️ 문서 내비게이션

    📂 00. Project Overview

    프로젝트의 심장! 목표(KPI), 범위(In/Out), 그리고 우리가 관리해야 할 리스크를 확인하세요. - 목표 및 전략 | 역할 정의(RBAC) | 가정 및 리스크

    📂 10. Domain Logic

    비즈니스의 규칙! 작업(WorkOrder)이 어떻게 흘러가고, 어떤 점검을 거치는지 정의합니다. - WorkOrder FSM | 체크리스트 정책

    📂 20. Product Spec

    사용자의 경험! 관리자, 팀장, 기사님이 마주할 실제 화면의 기능을 상세히 다룹니다. - 본사(Admin) 스펙 | 팀(Team Manager) 스펙 | 기사(Technician) 스펙

    📂 30. Technical Spec

    개발자를 위한 나침반! API 명세부터 CRUD 권한, 컴포넌트 설계까지 기술적인 약속을 담았습니다. - M-02 통합 구현 | API 가이드 | CRUD 매트릭스

    데이터의 뼈대! 테이블 간의 관계(ERD)와 각 필드가 의미하는 바를 한눈에 보여줍니다. - ERD (Mermaid)
